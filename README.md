# NanoRaider

NanoRaider is a prototype roguelike-inspired management game where your playstyle and decisions shape what kind of legacy your hero forges.
It draws inspiration from Melvor Idle, World of Warcraft, Slay the Spire, Stardew Valley, and Reigns.

## Try It

Play the latest web build here: [https://rissole.github.io/nanoraider/](https://rissole.github.io/nanoraider/)

## Run Locally

```bash
git clone git@github.com:rissole/nanoraider.git
cd nanoraider/client
npm install
npm run dev
```

Open the local URL shown by Vite (usually `http://localhost:5173`).

## Build

From `client/`:

```bash
npm run build
npm run preview
```

## Project Layout

- `client/` - frontend app (React + TypeScript + Vite)
- `design/` - evolving game design docs
